# AI-Driven Face Analysis System

## Overview

The **AI-Driven Face Analysis System** is a comprehensive, real-time platform that integrates multiple advanced AI features into one solution. It is designed to analyze facial data and provide functionalities such as:

* Emotion detection with real-time suggestions
* Liveness detection to prevent fake images
* Health monitoring using facial features
* Smart attendance recognition
* Face shape classification with personalized style recommendations
* Age and gender prediction

This system is suitable for applications in healthcare, security, education, and corporate environments, enhancing emotional well-being, health tracking, and secure user identification.

## Features

1. **Emotion Detection:** Detects happiness, sadness, anger, fear, disgust, surprise, and neutral emotions. Provides personalized suggestions or motivational quotes.
2. **Liveness Detection:** Verifies real human faces and prevents spoofing with AI-generated or fake images.
3. **Health Monitoring:** Analyzes facial signs like skin tone, eye condition, and stress indicators to provide basic health insights.
4. **Smart Attendance Tracking:** Automatically logs attendance by recognizing registered users in real-time.
5. **Face Shape Classification:** Identifies face shapes (oval, square, round, heart, pear, etc.) and gives style recommendations for hair, glasses, and beard.
6. **Age & Gender Prediction:** Estimates a personâ€™s age and gender accurately using deep learning models.

## Technology Stack

* **Programming Languages:** Python, PHP, JavaScript
* **Libraries/Frameworks:** OpenCV, TensorFlow/Keras, Flask/PHP integration, HTML/CSS/JS
* **Machine Learning Models:** CNN-based emotion, age, gender, and face shape classifiers
* **Database/File Storage:** JSON or local storage for logs

## Usage

1. **Live Camera Mode:** Open the live camera to detect emotions, monitor health, verify liveness, and track attendance in real-time.
2. **Upload Mode:** Upload an image for age, gender, emotion, or health analysis.
3. **Suggestions:** Personalized hairstyle, glasses, and motivational tips appear based on detected emotions or face shape.

## Installation

1. Clone the repository:

```bash
git clone <repo-url>
```

2. Install required Python packages:

```bash
pip install -r requirements.txt
```

3. Run the main script:

```bash
python main.py
```

4. Access the system via your web interface (PHP pages for frontend interaction).
